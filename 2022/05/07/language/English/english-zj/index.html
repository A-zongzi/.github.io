# 语法

# 准备工作

在认识语法前需要认识的内容

认识s、谓、o、p、attr、ad、c

# S

表示句子叙述的主体

graph LR
		A(句子叙述的主体) --> B(主动语态) --> C(发出动作)
		A --> D(被动语态) --> E(承受动作)

祈使句可以省略 S
eg.You must stop!→Stop！

# 谓

表示 S” 做什么” 或” 是什么”
eg.She speaks(谓语)a little French 框内的即为谓语部分

# O

表示动作的接受者
eg.He passedme(间 o)the book(直 o | 动作直接的接受者)

# P

表示 S"是什么样",系动词(和 S&P 组成完整句子)后
eg.The flower smellsgreat(P)

# attr

表示限定，修饰 n/pron...
eg.The placeI visited(后置定语)

# ad

表示状态
eg.This is goodenough(程度 ad)

# C

graph LR
A(补充S.O细节) --> B(SC)
A --> C(OC)

eg.Imangry(SC)

eg.He made mecry(OC)

# appo

表示地位相同
eg.Paris,the capitd of France,is a beautiful city

# 插入语

表示附加说明，本身不作句子的任何成分
eg.Strange(语气),there is nobody in the room

# 基础知识

简单句五大基本句型+谚语应用

# S+Vi(+ad)

# S+Vt+O

# S+V + 间 O + 直 O (双 O)

# S+lV+P(SC)

# S+V+O+OC
