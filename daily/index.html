# 关于缓存刷新时间过长问题的解决方法

  1. 在浏览器中打开 https://cdn.jsdelivr.net/gh/你的Github用户名/你的Github用户名.github.io/
  2. 将 https://cdn.jsdelivr.net/... 改为 https://purge.jsdelivr.net/... 后回车即可刷新缓存

# 解决了一个网站点击 QQ 无法跳转聊天窗口的问题

在添加 http://wpa.qq.com/msgrd?v=3&uin=你的qq&site=qq&menu=yes 的前提是

  1. 你的 qq 开启允许陌生人发起临时会话。
  2. https://shang.qq.com/v3/widget.html 开启 qq 推广通讯组件

# 解决了网站搜索的 BUG

bug 为搜索不显示跳转超链接文字
解决办法：添加环境变量
我的电脑 > 右键属性 > 高级设置 > 环境变量 > 新建 > 填写变量名称和变量值
变量名称 : HEXO_ALGOLIA_INDEXING_KEY
变量值：在 algolia 新建的有修改权限的 adminApikey
添加完在博客文件夹下点击 git bath here 输入 hexo algolia 就可以生成 index 上传到 algolia 进行搜索显示了

# 解决了一个让我头疼一天的 BUG

本来准备把 j 的笔记整理在网站上，结果发现流程图 html 显示不了，于是找来各种插件，各种教程试了一遍，昨日虽然用图片生成了但是和代码块冲突，所以今天早上顺着昨天的思路用上了 npm install hexo-filter-mermaid-diagrams --save 这个插件，改下配置文件即可，操作方法如下：

下载完 hexo-filter-mermaid-diagrams 后打开 Shoka 主题的 _config.yml 配置文件，找到 mermaid 的配置项 (没有的话添加)，并设置 enable: true ，如下所示：

如果还是不能正常显示的话，可以尝试以下操作：

  • 下载 mermaid.min.js 文件
    在 mermaid CDN 下载 mermaid.min.js 和 mermaid.min.js.map

  • 找到 themes/source/js 目录下的 js 存放文件夹处，将下载的 mermaid.min.js 和 mermaid.min.js.map 文件放进去

  • 找到 \themes\layout\partials 下的 layout.njk 添加
    {%- if theme.mermaid.enable %}
    <script src="https://cdn.jsdelivr.net/npm/mermaid@8/dist/mermaid.min.js">
    </script>
    {%- endif %}
    效果如下～～（由于是用 script 标签直接加载 js，所以有一些慢，可以适当刷新等待）~~：

# 下载网页视频的技能

首先用谷歌浏览器打开要下载的视频，然后按下 F12 ，点击 Network 一栏，可以看到流量请求的一些项目， Filter 框框里面输入 http , 然后会显示播放的一个脚本，点击脚本后旁边选择 Response ， crtl+F 搜索 video 或者 src ，找到 video 盒子下的一个 src 链接，复制到网页打开即可在线播放或者下载。
